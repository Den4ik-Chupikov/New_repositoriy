$ git status - proverka papki na nalichie faylov

$ git add (imya faayla)	- dobavlenie fayla v soxranenie, dobavlenie v index. Changes to be committed.

$ git commit -m" "  - soxranyaem nakoplennnue izmeneniya v fayle. v kovuchki, chto planiruem dobavit

$ git log - proverka soxraneniya, chto soxranili.Spisok "commit"
